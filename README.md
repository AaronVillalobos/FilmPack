# FilmPack
An iOS app where collections of instant-photos (ex. Polaroids) can be published to a feed for personal viewing.

## Tech Stack
* Swift/SwiftUI

## How it Works
1. Create a new 'FilmPack' by tapping the '+' button
2. Choose up to 8 instant photos from your device's photo library
3. Add a title and an optional caption, then publish
4. See your newly created 'FilmPack' in your personal feed

*Note: This app and its contents is not available on the App Store. It is intended for the developer's personal use and technical exploration (and for the love of Polaroids).
